# Тесты для лабораторной работы №01

## Входные данные
double xQ, yQ, xR, yR, xP, yP

## Выходные данные
Целое 0, 1 или 2 

## Позитивные тесты:
- 01 - точка над прямой;
- 02 - точка на прямой;
- 03 - точка под прямой;
- 04 - прямая вертикальна;

## Негативные тесты:
- 01 - вместо некоторых из координат - буквы;
- 02 - точки, задающие прямую, совпадают;
